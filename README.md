# 📊 Workflow Performance & Bottleneck Analysis (DocuWare-Inspired)

## 📌 Project Overview

This project analyzes document workflow data to identify **process bottlenecks, step delays, and workflow performance trends**.
The analysis simulates workflow logs similar to those generated by document management systems such as **DocuWare**.

The goal of the project is to demonstrate how **data analytics can improve operational efficiency** by identifying workflow steps that slow down document processing.

---

## 🎯 Objectives

* Analyze workflow logs to measure **processing time between workflow steps**
* Identify **bottleneck steps** that delay document processing
* Visualize workflow performance using **charts and dashboards**
* Build an **interactive analytics dashboard** using Streamlit

## 🔍 Dataset Description

The dataset represents workflow logs with the following fields:

| Column          | Description                           |
| --------------- | ------------------------------------- |
| workflow_ID     | Unique workflow identifier            |
| step_type       | Type of workflow step                 |
| step_order      | Order of the step in the workflow     |
| step_name       | Name of the workflow step             |
| completion_time | Timestamp when the step was completed |

Using this information, the project calculates **processing time between workflow steps**.
